The single load-bearing belief in the current platform thesis that is least supported by the corpus is the **genotype-informed-supplement-workflow's implicit claim that the five-step closed-loop pipeline (genotype → compound selection → produce/source → Tier 2 batch QC → calibrated dose → biomarker tracking) is n=1-operational for every intervention class the platform recommends.** The workflow has been instantiated at n=1 for zero interventions — it exists as a documented specification, not as a field-tested protocol. The Q141K butyrate worked example is the only worked example; it documents a Tier 2 assay gap (comp-038 YELLOW — no home colorimetric butyrate assay exists) that breaks the "calibrate once at Tier 3, track batches at Tier 2" discipline at step 4 for the very intervention the worked example prioritizes. The cordycepin Tier 2 diazo-coupling assay is speculative (§1.28, not yet validated). The EGT Ellman's reagent Tier 2 assay is well-established but no multi-operator reproducibility data exists. The GLPP Tier 2 phenol-sulfuric assay measures total polysaccharide, not GLPP specifically. Every Tier 2 assay in the pipeline has a documented gap or caveat; none has been validated in a multi-user setting. The workflow's five steps are individually specified but have never been executed end-to-end by any user, including Brian. H09 (Community Fermentation Reliability) is the platform-level test of the production side; no equivalent falsification card exists for the *workflow integration* side — the claim that a user can genotype themselves, select a compound, produce or source it, verify the batch potency at Tier 2, calibrate the dose, and track biomarkers in a closed loop. The workflow is specified; the Tier 2 assays are specified; the Tier 3 anchors are specified; the self-experiment protocol is specified. Every component exists. No one has run the full loop. The single most informative near-term experiment is not a new wet-lab gate — it's a **dry run of the workflow end-to-end on one compound where all Tier 2 infrastructure is already validated** (ergothioneine — Ellman's reagent is well-established, *P. citrinopileatus* cultivation is home-feasible, Tier 3 HILIC-HPLC anchor exists, OCTN1 variant is genotypable via clinical PCR). Cost: ~$500 (genotyping + grow kit + Ellman's reagent + one Tier 3 HPLC anchor). Time: 8–12 weeks (cultivation cycle + assay + biomarker tracking). This is a faster path to surfacing workflow failures than waiting for the §1.9 Ward 1995 dual-cassette strain to exist.

*Anchor: `genotype-informed-supplement-workflow.md` §"Why this exists" claims the workflow blocks two failure modes (silent underdosing and genotype-blind selection) and §"How this fits with H09" claims the workflow survives even if H09 fails. The Q141K worked example's step 4 uses stool SCFA as a Tier 2 proxy that comp-038 explicitly says is not a direct potency measurement. `quantification-ladder.md` §"The operational pattern" requires "calibrate once at Tier 3, track batches at Tier 2" — and for butyrate, Tier 2 tracking does not exist. The workflow's design is sound; its field-readiness for butyrate specifically (the intervention its only worked example uses) has not been verified.*
