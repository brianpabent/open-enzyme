# Exertion-challenge test (two-phase: no colchicine cover → colchicine cover, n=1).

1. **Exertion-challenge test (two-phase: no colchicine cover → colchicine cover, n=1).** Cost: ~$50 (UASure strips); pre-existing UASure meter. Time: 2 × 2-hour sessions, ~4 weeks apart. Decides: whether exertion/fatigue triggers flares via metabolic-overload (UA rises post-exertion → mechanism #5 in `mechanical-flare-triggers.md`) or via mechanical shedding / mechanotransduction / DAMP / effusion (UA unchanged but flare occurs → mechanisms #1–4). This is the $50 experiment that discriminates between the two classes of explanation for the single most common non-dietary flare trigger in the Li XD 2012 Qingdao n=1,713 cohort. Rationale detailed in Connection 1 above. Pre-register the prediction before running: if UA unchanged + flare → mechanical; if UA rises + flare → metabolic-overload. Cross-references: `mechanical-flare-triggers.md` §"Open question 7," `fructose-connection.md`, `validation-experiments.md §3.10` (fructose challenge — consider replacing or supplementing with this).
