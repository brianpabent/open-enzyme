1. **The multi-track coverage map in `gout-pathophysiology.md` presents ABCG2 coverage from the koji track as "Lactoferrin → TNFα suppression → ABCG2 derepression" in the same table as direct transporter modulation claims (cordycepin → URAT1, GLPP → GLUT9/OAT1, astilbin → XO) — but the evidence tiers are fundamentally different.** Locations: `gout-pathophysiology.md` §"Multi-track urate transporter coverage" table. Analysis: The table uses a unified format where each cell names a compound and mechanism. Cordycepin's URAT1 effect is Animal Model (direct). GLPP's GLUT9/OAT1 effects are Animal Model (direct). Astilbin's XO inhibition is Animal Model (direct). Lactoferrin's ABCG2 effect is **Mechanistic Extrapolation** (lactoferrin → TNFα suppression is documented; TNFα → ABCG2 suppression is documented; the composed chain lactoferrin → ↑ABCG2 has no direct experimental confirmation — it is a two-step inference). The table's footnote acknowledges this ("Mechanistic Extrapolation — indirect, not direct transporter modulation"), but the visual presentation in a unified coverage table risks overstating the ABCG2 coverage claim. A reader scanning the table will see "ABCG2: covered by koji track" without absorbing the evidence-tier distinction. This is a presentation contradiction, not a factual error — the footnote is honest, but the table format undermines the honesty.
   - *Suggested Action:* Add a visual indicator to the ABCG2 cell (e.g., "(indirect, ME)" or a distinct background shading) that distinguishes Mechanistic Extrapolation claims from direct Animal Model claims. Alternatively, split the ABCG2 row into "ABCG2 (direct modulation)" and "ABCG2 (indirect derepression)" to make the evidence tier visible at the row level rather than buried in a footnote.
